OBS 적용 방법

  1. 치지직 채팅 오버레이 URL을 준비합니다
  2. OBS에서 브라우저 소스를 추가합니다
  3. URL에 치지직 오버레이 주소를 입력합니다
  4. 사용자 지정 CSS 란에 white-tiger-overlay.css 내용을 붙여넣습니다
  5. 크기를 조절하고 확인을 누릅니다
